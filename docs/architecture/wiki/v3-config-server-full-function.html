Review checklist

  • Config IO only through V3ConfigStore.
  • Manifest contains declarations/indexes; no selected route/provider, resolved secret, or request decision.
  • All enabled listeners bind before any serving task starts.
  • Invalid HTTP input enters Error01-06 before Runtime; no synthetic payload.
  • Only /v1/responses executes P6; other protocols remain explicit not_implemented.
  • Relay, continuation runtime, Chat Process, Hub cutover, P6 deletion, V2/live/global install remain out of scope.
