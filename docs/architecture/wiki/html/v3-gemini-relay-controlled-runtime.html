Node logic cards

Gemini codec owns Gemini semantics

URL/model, candidates, functionCall, finishReason, and stream terminal rules stay in Gemini codec/runtime.

Server consumes binding only

Server dispatches to the Gemini runtime owner and transports typed output; it does not parse Gemini response semantics.

Malformed streams fail explicitly

Malformed JSON, EOF without terminal, and post-terminal frames are provider/runtime errors, not hidden successes.
