Node logic cards

Normalization is not governance

Codecs map protocols and preserve invalid tool identity shapes for Chat Process rejection or governance.

Compat is not fallback

Provider compat nodes perform provider micro-adjustments only; no route, model, tool, or fallback policy.

Chat Process owns tool semantics

Tool pairing, duplicate identity, orphan output, servertool, stopless, and continuation belong to Req04/Resp03.
