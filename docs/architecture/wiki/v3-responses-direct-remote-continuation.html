Review checklist

  • One Runtime kernel core and one response exit serve both turns.
  • Continuation loads only at Req03 and resolves the immutable exact pin only at Req06.
  • No Virtual Router re-entry, target-local reselection, Relay, or local materialization occurs.
  • Resp04 observes provider semantics and commits, replaces, or releases the matching locator.
  • Provider/client payloads contain no session, auth, owner, capability, or routing control truth.
  • JSON and SSE controlled HTTP replay are required; current 5555 replay remains pending.
